# Lab Sample Tracker

A Node.js/Express laboratory sample tracking application for managing sample registration, vessel details, storage records, and retrieval workflows.

## Overview

Lab Sample Tracker is a practical laboratory operations application designed to improve how petroleum laboratory samples are registered, stored, searched, and retrieved.

The project is based on a real laboratory workflow where sample information needs to be traceable, structured, and easy to retrieve.

## Features

- Register laboratory samples
- Capture vessel and client details
- Record sample source, storage, and retrieval information
- View sample records through dashboard pages
- Organize sample data using backend routes and views
- Support a more structured replacement for manual sample register books

## Technology Stack

- Node.js
- Express.js
- EJS
- CSS
- JavaScript
- SQLite/database setup
- Git and GitHub
